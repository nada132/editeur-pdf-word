Installer sur mon ordinateur

Utilise ce logiciel directement depuis ton Bureau Windows, sans navigateur.

⬇ Télécharger le logiciel (.zip)
  1. 1 Clique sur le bouton ci-dessus et télécharge le ZIP
  2. 2 Extrais le dossier (clic droit → Extraire tout)
  3. 3 Double-clique sur DEMARRER.bat pour lancer
  4. 4 Pour la barre des tâches : clic droit sur DEMARRER.bat → Épingler à la barre des tâches

⚠️ Node.js est requis : télécharger Node.js (gratuit, installation en 1 clic)
